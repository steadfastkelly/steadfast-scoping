# ImportFeedbackBanner

## Purpose

Single shared feedback surface for upload, validation, and import messaging.

## Responsibilities

- display success, warning, or error tone
- show title and descriptive copy
- support upload-stage errors
- support validation-state pass or warning messages
- support import-stage failure messages

## Inputs

- tone
- title
- description

## Reuse Rule

- use instead of separate alert and banner components
